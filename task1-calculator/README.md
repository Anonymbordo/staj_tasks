# Task 1 - Gelismis Hesap Makinesi

Java 17 ile yazilmis komut satiri uygulamasi ve iPhone tarzinda gelismis tarayici arayuzu bulunan hesap makinesi.

## Ozellikler

- Toplama, cikarma, carpma, bolme
- Karekok alma ve us alma
- Parantezli ifade hesaplama
- Islem onceligi destegi
- Trigonometrik fonksiyonlar: sin, cos, tan
- Logaritmik fonksiyonlar: ln, log
- Sabitler: pi ve e
- Derece/radyan modu
- Hafiza islemleri: MC, MR, M+, M-
- Son cevap kullanimi: Ans
- Islem gecmisi paneli
- Klavye destegi
- Sifira bolme ve gecersiz ifade hata kontrolu

## Komut Satiri Calistirma

```bash
javac -d out src/main/java/com/murathan/task1/Main.java
java -cp out com.murathan.task1.Main
```

## Arayuz Calistirma

`index.html` dosyasini tarayicida acabilirsiniz.
