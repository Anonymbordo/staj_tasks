# Task 3 - Stream API

Java Stream API kullanilarak hazirlanmis sayisal ve alfabetik ornekler.

## Istenen Ornekler

- 1'den 100'e kadar sayilari listeye alma
- Hem 2 hem 4 ile bolunebilen sayilarin adedini bulma
- Tek sayilarin adet ve toplamlarini bulma
- Cift sayilarin adet ve toplamlarini bulma

## Ek Ornekler

- 5 ile bolunebilen sayilari listeleme
- Sayilarin karesini alma
- Alfabetik isim filtreleme
- Isimleri alfabetik siralama
- Harfe gore gruplama

## Calistirma

```bash
javac -d out src/main/java/com/murathan/task3/Main.java
java -cp out com.murathan.task3.Main
```
