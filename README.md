# Staj Görevleri

Bu repository, staj sürecinde istenilen Java ve Spring Boot görevlerini içermektedir. Her görev ayrı klasör altında hazırlanmıştır ve ilgili klasörlerde çalıştırma komutları ile açıklamalar bulunmaktadır.

## Görev Listesi

| Görev | Açıklama | Klasör |
| --- | --- | --- |
| Task 1 | Komut satırında çalışan hesap makinesi ve gelişmiş hesap makinesi arayüzü | [task1-calculator](./task1-calculator) |
| Task 2 | Sayı tahmin oyunu ve matematik parkur oyunu | [task2-number-guessing-game](./task2-number-guessing-game) |
| Task 3 | Java Stream API ile sayısal ve alfabetik örnekler | [task3-stream-api](./task3-stream-api) |
| Task 4 | Spring Boot ve PostgreSQL ile kullanıcı CRUD uygulaması | [task4-spring-boot-postgresql](./task4-spring-boot-postgresql) |
| Task 5 | Spring Boot ile kütüphane yönetimi CRUD uygulaması | [task5-library-management](./task5-library-management) |

- Task 1, Task 2 ve Task 3 Java temel konularını ve OOP/Stream API kullanımını kapsar.

- Her task kendi klasörü altında bağımsız şekilde incelenebilir.
